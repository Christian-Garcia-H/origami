Origami

La libreria de css que facilita tu trabajo

¿Qué puedes hacer con la librería Origami?

Esta biblioteca de CSS ha sido diseñada para acelerar el desarrollo web mediante un sistema de clases utilitarias que permiten aplicar estilos directamente en el código HTML de manera rápida y eficiente. Su estructura modular optimiza la gestión de propiedades como márgenes, espaciados, tipografía, colores y alineación, reduciendo el tiempo de desarrollo sin sacrificar la calidad del diseño.

A diferencia de otras soluciones que abstraen por completo la escritura de estilos, esta biblioteca permite al desarrollador mantener y fortalecer sus habilidades de CSS, ya que la personalización avanzada se logra a través de hojas de estilo adicionales. Gracias a esta combinación, el desarrollador tiene control total sobre la apariencia del proyecto, integrando paletas de colores personalizadas, fuentes tipográficasy otros ajustes específicos que se adapten a la identidad del proyecto.

Para usar la librería Origami en tu proyecto, incluye el siguiente código en la sección <head> de tu archivo HTML.

 <link rel="stylesheet" href="https://christian-garcia-h.github.io/origami/Origami.css">

Colores

Todos los colores de esta librería y los colores personalizados se pueden usar tanto para background como para color de texto. Una forma fácil de identificarlos es que las clases que empiezan con la letra c corresponden a color de texto y las que empiezan con bg corresponden a background.

Ejemplo:

para color de fondo azul:

 bg-blue

para color de texto azul:

 c-blue

Colores primarios

Los colores primarios disponibles son tres:

#0000ffff

#00ff00ff

#ff0000ff

Estos colores se pueden usar con las clases utilitarias:

 bg-blue bg-green bg-red

Colores neutros

Tambien tiene tres colores neutros:

#000000ff

#808080ff

#ffffffff

Estos colores se pueden usar con las clases utilitarias:

 bg-black bg-grey bg-white

Otros colores

Hay una línea de otros colores que también son muy útiles:

#ffff00ff 

 bg-yellow

Colores personalizados

En total la versión 1.1 de esta librería tiene 7 colores básicos integrados, pero se pueden añadir 8 colores mas por medio de un archivo CSS adicional en el cual se defina una paleta de colores personalizada con la siguiente sintaxis: 

 :root {
                        --one: #d00000ff;
                        --two: #fca311ff;
                        --three: #6a040fff;
                        --four: #6b705cff;
                        --five: #90caf9ff;
                        --six: #42a5f5ff;
                        --seven: #1976d2ff;
                        --eight: #f4d58dff;
                    }

A estos colores puedes acceder fácilmente usando las clases utilitarias desde tu documento HTML:

 bg-one bg-two bg-three ....

Para utilizar colores personalizados, asegúrate de vincular tu archivo CSS al documento HTML después de importar la librería Origami. Si lo vinculas antes, solo se aplicarán los colores predeterminados de la librería.

Box model

¿Qué ofrece esta libreria?

Esta librería CSS está diseñada para optimizar el desarrollo web mediante el uso de box-sizing: border-box;, asegurando que el tamaño de los elementos se mantenga consistente al incluir bordes y relleno dentro de sus dimensiones definidas. Además, ofrece un conjunto de utilidades para aplicar de forma rápida y eficiente bordes, márgenes,padding, así como ajustes de width y height, permitiendo un diseño más flexible y estructurado.

padding

Con clases predefinidas, es posible aplicar distintos niveles de padding de forma flexible, adaptándose a las necesidades de diseño y optimizando el flujo de trabajo en el desarrollo web.

Cada clase sigue una escala progresiva, utilizando valores en rem para mantener accesibilidad y flexibilidad en cualquier resolución de pantalla.

Clases Generales (p-*)

Las clases de padding generales aplican el mismo espaciado en todos los lados del elemento:

<div class="p-3">Contenido con padding uniforme</div>

Tenemos opciones desde p-1 hasta p-16 lo que permite ajustar el padding desde 4px hasta 64px en una escala progresiva.

Clases Direccionales (pt-*, pb-*, pl-*, pr-*, px-*, py-*)

Puedes usar la dirección en la que se aplicara el padding junto a la proporción de padding antes descrita pt-* (arriba), pb-* (abajo), pl-* (izquierda), pr-* (derecha), px-* (horizontal) y py-* (vertical).

Ejemplo padding a todas las direcciones:

 p-1 p-2 p-3 ... p-16

Ejemplo padding top:

 pt-1 pt-2 pt-3 ... pt-16

Ejemplo padding bottom:

 pb-1 pb-2 pb-3 ... pb-16

Ejemplo padding left:

 pl-1 pl-2 pl-3 ... pl-16

Ejemplo padding rigth:

 pr-1 pr-2 pr-3 ... pr-16

Ejemplo de padding horizontal:

 px-1 px-2 px-3 ... px-16

Ejemplo de padding vertical:

 py-1 py-2 py-3 ... py-16

margin

Las clases predefinidas de margen permiten ajustar el espaciado entre elementos de manera flexible, facilitando un diseño más ordenado y una mejor distribución del contenido.

Siguiendo la misma idea de escala progresiva que en padding, cada clase utiliza valores en rem, asegurando coherencia, accesibilidad y adaptabilidad en cualquier resolución de pantalla. Esto simplifica el control del diseño y agiliza el flujo de trabajo en el desarrollo web.

La única diferencia es el uso del valor auto, disponible en los márgenes para centrar elementos de manera automática dentro de su contenedor. Esto brinda mayor control sobre la alineación y facilita la creación de diseños responsivos.

Clases Generales (m-*)

Las clases de margin generales aplican el mismo espaciado en todos los lados del elemento:

<div class="m-3">Contenido con padding uniforme</div>

Tenemos opciones desde m-1 hasta m-16 lo que permite ajustar el margin desde 4px hasta 64px en una escala progresiva ademas del valor auto.

Clases Direccionales (mt-*, mb-*, ml-*, mr-*, mx-*, my-*)

Puedes usar la dirección en la que se aplicara el padding junto a la proporción de padding antes descrita mt-* (arriba), mb-* (abajo), ml-* (izquierda), mr-* (derecha), mx-* (horizontal) y my-* (vertical).

Ejemplo margin a todas las direcciones:

 m-1 m-2 m-3 ... m-16 m-auto

Ejemplo margin top:

 mt-1 mt-2 mt-3 ... mt-16 mt-auto

Ejemplo margin bottom:

 mb-1 mb-2 mb-3 ... mb-16 mb-auto

Ejemplo margin left:

 ml-1 ml-2 ml-3 ... ml-16 ml-auto

Ejemplo margin rigth:

 mr-1 mr-2 mr-3 ... mr-16 mr-auto

Ejemplo de margin horizontal:

 mx-1 mx-2 mx-3 ... mx-16 mx-auto

Ejemplo de margin vertical:

 my-1 my-2 my-3 ... my-16 my-auto

width

El sistema de clases utilitarias para definir el ancho de los elementos sigue una escala progresiva que mantiene coherencia y flexibilidad en el diseño. Para los valores más pequeños, los incrementos son de 4px, abarcando desde w-1 hasta w-6. A partir de w-8 y hasta w-16, los incrementos aumentan a 8px, mientras que desde w-20 en adelante, hasta w-90, los valores crecen en pasos de 16px.

Esta estructura permite ajustar el ancho de los elementos de forma intuitiva, asegurando una mejor distribución del contenido y una mayor adaptabilidad en diferentes resoluciones de pantalla.

w-1 hasta w-6

 w-1 w-2 w-3 w-4 w-5 w-6

w-8 y hasta w-16

 w-8 w-10 w-12 w-14 w-16

w-20 hasta w-90

 w-20 w-24 w-28 w-32 ... w-82 w-86 w-90

también puedes usar porcentajes y width adaptable al contenido

width con porcentajes

Las clases de ancho basado en porcentajes ofrecen una solución flexible y adaptable para el diseño responsivo. Al utilizar valores relativos en lugar de fijos, los elementos pueden ajustarse dinámicamente al tamaño de su contenedor, asegurando una mejor experiencia en distintos tamaños de pantalla.

los porcentajes disponibles son: 10% 25% 33.3% 50% 66.6% 75% 100%

 w-10p w-25p w-33p w-50p w-66p w-75p w-100p

width adactable al contenido

Las clases de ancho que se ajustan al contenido permiten que los elementos ocupen solo el espacio necesario según su contenido, sin forzar dimensiones fijas. Esto es ideal para elementos como botones, etiquetas o contenedores que no deben ocupar más espacio del requerido. El uso de valores como fit-content, max-content y min-content ofrece flexibilidad en el diseño, asegurando que los elementos se adapten de manera óptima a diferentes longitudes de texto o contenido dinámico. Esto mejora la accesibilidad y mantiene una distribución eficiente en cualquier entorno responsivo.

los valores disponibles son:

 w-fit w-max w-min

width auto

Tan solo usa el valor auto para la propiedad width 

 w-auto

height de ventana gráfica

Las clases de altura basadas en la ventana gráfica (vh) permiten definir elementos con una altura relativa al tamaño total de la pantalla del usuario.

Utilizando:

 h-25vh, h-50vh, h-75vh y h-100vh
Es posible establecer alturas equivalentes al 25%, 50%, 75% y 100% del viewport, respectivamente. Esto resulta útil para secciones que necesitan ajustarse dinámicamente al tamaño de la pantalla, como banners, secciones de presentación o contenedores de fondo a pantalla completa.

border

En esta librería, los bordes se controlan con la clase b, que aplica un borde sólido de 1px por defecto. Si se desea eliminar el borde, se usa b-none. También es posible definir bordes en lados específicos con: 

 b-t  b-r  b-b  b-l
aplicándolos solo en la parte superior, derecha, inferior o izquierda respectivamente.

El grosor del borde se gestiona con:

 b-1  b-2  b-3  b-4

donde cada número representa el grosor en píxeles. Para cambiar el estilo del borde, se pueden usar:

 b-solid  b-dashed  b-dotted  b-double
dependiendo de si se requiere un borde sólido, punteado, rayado o doble.

El color del borde se maneja en dos categorías. Los colores predefinidos en la librería se identifican con nombres como 

 b-green  b-blue  b-red  b-black  b-grey  b-white ..

los cuales usan variables CSS internas (--green, --blue, etc.). Para opciones personalizadas, existen:

 b-one  b-two  b-three  b-four ... b-seven  b-eight

que permiten adaptar la paleta de colores según la necesidad del usuario.

Para aplicar bordes redondeados, se usa br para un redondeo general. Se pueden utilizar variantes como br-none para eliminar el redondeo, para diferentes niveles de curvatura:

 br-sm  br-md  br-lg

y br-full para un borde completamente circular. También existen clases para redondear esquinas específicas: 

 br-t-sm  br-r-sm  br-b-sm  br-l-sm

Todas estas clases pueden combinarse entre sí, permitiendo una personalización flexible sin necesidad de escribir CSS adicional.

Fuentes

¿Qué tamaños de fuente ofrece la libreria?

La librería ofrece una serie de tamaños de letra predefinidos que permiten ajustar fácilmente la tipografía sin necesidad de escribir CSS adicional. El tamaño base recomendado para el texto es fs-16, equivalente a 16px, ideal para párrafos y contenido general. Para textos más pequeños, se pueden usar fs-10, fs-12 o fs-14, que funcionan bien en etiquetas, notas o detalles secundarios. Para subtítulos y elementos destacados, fs-20 y fs-24 ofrecen una buena relación entre legibilidad y jerarquía visual. Los títulos y encabezados pueden usar fs-30, fs-36 o fs-48, dependiendo del énfasis que se quiera dar. En secciones grandes o elementos de impacto, como hero sections, se recomienda fs-60 o fs-72, aunque en dispositivos móviles estos tamaños se ajustan automáticamente para mejorar la adaptabilidad. Todos estos valores siguen una escala tipográfica que mantiene la armonía visual en diferentes tamaños de pantalla.

fuentes disponibles

Por el momento, la librería solo incluye la fuente Krub de Google en sus variantes estándar (400) y bold (700). Esto garantiza un diseño limpio y consistente, manteniendo un equilibrio entre legibilidad y estilo. Para aplicar estas fuentes, se pueden usar las siguientes clases: 

 krub-regular /*Aplica Krub en su variante estándar (400)*/ 
 krub-bold /*Aplica Krub en su variante negrita (700)*/

Para mayor flexibilidad, se recomienda crear un archivo CSS aparte donde se puedan definir fuentes personalizadas según las necesidades del proyecto. En futuras versiones, se podrían agregar más opciones tipográficas de manera nativa.

alineacion de texto

La librería incluye clases utilitarias para alinear y modificar el estilo del texto de manera sencilla. Para la alineación, se pueden utilizar:

 t-align-c /*para centrar el texto*/ 
 t-align-r /*para alinearlo a la derecha*/ 
 t-align-l /*para alinearlo a la izquierda*/

Además, la clase t-dn elimina cualquier subrayado o decoración del texto, útil para enlaces o elementos donde se quiera un estilo más limpio.

Diccionario de términos

¿Qué es una clase utilitaria?

Una clase utilitaria en CSS es una clase predefinida con una única responsabilidad: aplicar un estilo específico a un elemento sin necesidad de escribir CSS adicional. Estas clases permiten modificar rápidamente propiedades como márgenes, colores, tipografía, alineaciones y más, directamente en el código HTML. A diferencia de las clases tradicionales, que agrupan múltiples estilos dentro de una misma regla, las clases utilitarias están diseñadas para ser pequeñas, modulares y reutilizables, lo que facilita la creación de interfaces de manera eficiente y consistente.

Ejemplo:

podemos usar la siguiente clase utilitaria dentro de nuestro Código HTML

bg-yellow

y esto hará referencia a esta clase de la librería, de esta forma ahorraremos tiempo al escribir Código

 .bg-yellow {
                        background-color: var(--yellow);
                    }
